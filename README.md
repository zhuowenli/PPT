#PPT

Powered by: [nodePPT](https://github.com/ksky521/nodePPT)

### 安装

    npm install -g nodeppt

### 启用

clone项目到本地

    git clone git@github.com:zhuowenli/PPT.git

切换到子目录

    cd Front-End-Architecture

执行`nodeppt start`，浏览器会自动打开该PPT。

    nodeppt start
